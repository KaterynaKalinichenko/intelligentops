Hello!

This script will expand the number of instances you specified, create a security group and add your ssh key to the server. :)

1. Install Terraform on the server/local pc;
		https://www.terraform.io/downloads.html
		
2. Copy files to any folder;
3. Modify files - variables.tf according to the comments;
4. Run the following commands in a terminal or console:
		terraform init
		terraform apply
		
Enjoy :D
